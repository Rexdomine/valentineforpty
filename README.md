# Valentine Page

## What to drop in
- `assets/song.mp3`
- `assets/lyrics.lrc` (LRC timestamps)
- `assets/roses.gif`

## Run locally
From this folder:

```bash
python3 -m http.server 5173
```

Open:
- http://localhost:5173/index.html

Notes:
- Autoplay usually works because the user clicks "Yes" first, but some browsers still require one tap on the play button.
